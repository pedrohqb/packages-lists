2. Instalar dotfiles (configuração de fonte do Evolution): curl -s https://raw.githubusercontent.com/pedrohqb/dotfiles/refs/heads/main/instalar-dotfiles.sh | bash

3. Instalar aplicações (brew): brew install docx2txt graphicsmagick ocrmypdf onedrive-cli tesseract-lang yt-dlp

4. Instalar contêiner com aplicativos para advocacia (distrobox): distrobox-assemble create --file https://raw.githubusercontent.com/pedrohqb/distrobox-adv-br/refs/heads/main/distrobox-adv-br

5. Instalar contêiner com SoftMaker Office (distrobox): distrobox-assemble create --file https://raw.githubusercontent.com/pedrohqb/packages-lists/refs/heads/main/distrobox-assemble-files/softmaker-office

6. Instalar jogos (flatpak): ujust install-gaming-flatpaks

7. Instalar jogos adicionais (flatpak): flatpak install com.github.keriew.augustus com.play0ad.zeroad io.openrct2.OpenRCT2 org.libretro.RetroArch org.prismlauncher.PrismLauncher
