# Fedora KDE Plasma

1. Ajustar repositórios: habilitar Flathub, desabilitar Fedora Flatpaks e habilitar RPM Fusion Steam

2. Instalar virtualização (dnf): sudo dnf install @virtualization

3. Instalar aplicações (dnf): sudo dnf install audacity bibletime btrfs-assistant chromium distrobox evolution evolution-ews fastfetch flatseal gh gimp gimp-help-pt_BR GraphicsMagick k3b kate keepassxc langpacks-pt_BR ocrmypdf pdfarranger printer-driver-brlaser qbittorrent qpwgraph rclone rpm-build shrinkpdf snapper tesseract tesseract-langpack-eng tesseract-langpack-por yt-dlp

3. Instalar aplicações (flatpak): sudo flatpak install com.microsoft.Edge com.obsproject.Studio com.spotify.Client com.usebottles.bottles de.haeckerfelix.Shortwave net.cozic.joplin_desktop org.ferdium.Ferdium org.kde.kdenlive org.signal.Signal org.telegram.desktop org.verapdf.veraPDF org.videolan.VLC us.zoom.Zoom

4. Instalar jogos (dnf): sudo dnf install 0ad astromenace dolphin-emu dosbox-staging extremetuxracer gamemode gamescope gnuchess goverlay hedgewars knights kolf lutris mangohud retroarch steam supertux supertuxkart teeworlds warsow warzone2100 wesnoth xonotic

5. Instalar jogos (flatpak): sudo flatpak install com.github.keriew.augustus com.heroicgameslauncher.hgl io.openrct2.OpenRCT2 net.davidotek.pupgui2 net.openra.OpenRA net.veloren.airshipper org.prismlauncher.PrismLauncher

6. Instalar avulsos: lacuna-webpki mscorefonts2 pdfsam pje-office safesignidentityclient softplan-websigner
